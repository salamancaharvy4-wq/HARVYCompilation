# Activities

## Activity 1

Pokedex Generation Info System is a Laravel web activity that displays Pokemon by generation, showing starters and regional Pokemon lists with details such as type, ability, habitat, and evolution. It also includes a management page for uploading and deleting Pokemon images.

## Activity 2

Laundry Service Package CRUD is a Laravel activity that manages laundry service records through full CRUD operations, validation, search, and pagination.

## Activity 3

My Custom Input Form System is a Laravel activity focused on building and validating custom web forms. It includes a workshop enrollment form and an expense report form, both using input validation, custom error messages, old input retention, and success feedback after submission.
